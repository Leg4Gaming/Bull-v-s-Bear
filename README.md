# 🎮 Bull vs Bear

**Bull vs Bear** is a real-time multiplayer trading game that teaches financial literacy through fast-paced, news-driven gameplay.  
Trade stocks, crypto, bonds, and ETFs while reacting to live market events and competing for the highest risk-adjusted returns.

👉 **Learn markets by playing, not memorizing.**

---

## 🛠 Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS  
- **Backend:** Node.js, Express, Socket.IO  
- **AI:** Google Gemini (with smart fallback)  
- **State Management:** Zustand  

---

## 🎯 How It Works

1. Read breaking financial news  
2. Trade affected assets in real time  
3. Manage risk & diversify your portfolio  
4. Get AI feedback on your strategy  
5. Compete on the live leaderboard
